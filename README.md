##Todo-list

---
## Project Overview:
The **Todo Practice App** is a simple checklist application built using **HTML, CSS, and JavaScript**.  
It allows users to add tasks to a list and manage daily activities easily.

This project is perfect for beginners to understand **DOM manipulation**, **event handling**, and basic front-end structure.

---

## Features:
- Add tasks using an input field.
- Display tasks in a checklist format.
- Simple and clean user interface.
- Beginner-friendly code structure.
